# Project "beam_calc"
Wooden beam calculator

## Module features
- calculate loads:
  - set slab or roof slope;
  - set snow load;
  - set roof structural layers;
- calculate beam:
  - set beam loads;
  - set beam span;
  - set beam profile (profile width, height e.t.c.);

## Dependencies
- React;
- ReactDOM;
- Bootstrap;
